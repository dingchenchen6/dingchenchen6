# Chenchen Ding | СИЂТЎеТЎе

**Peking University - IIASA International Joint Postdoctoral Fellow**

Chenchen Ding (СИЂТЎеТЎе) is a biodiversity researcher working on **conservation biology**, **global change biology**, and **biogeography**. Her research focuses on how **land-use change** and **climate change** affect **biodiversity**, **species distributions**, **biodiversity change**, **community structure**, **ecosystem services**, **extinction risk**, and **conservation priority identification**.
